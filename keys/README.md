# Публичные SSH-ключи для доступа к серверам

Сюда кладётся публичный ключ, который скрипт `scripts/setup-head-ubuntu.sh` прописывает новому пользователю на головной VM.

## Что сделать

1. Создайте в этой папке файл **`deploy.pub`** с одной строкой — содержимым вашего публичного ключа.

   Например, скопируйте с вашей машины:
   ```bash
   cat ~/.ssh/id_ed25519.pub
   ```
   Вставьте эту строку в `keys/deploy.pub` и закоммитьте в репозиторий.

2. Публичные ключи (`.pub`) можно хранить в репе — это безопасно. В репозиторий **не** должны попадать приватные ключи (`id_ed25519`, `id_rsa` без `.pub`).

Файл `deploy.pub.example` — пример формата (подставьте свой ключ в `deploy.pub`).
